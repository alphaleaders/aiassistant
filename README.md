# Alpha Planner 📋

Telegram бот-планировщик дел с AI-секретарём (Groq) и веб-интерфейсом (Mini App).

## Архитектура

```
alpha-planner/
├── index.js                    # Точка входа — запуск бота, сервера, cron
├── .env                        # Конфигурация (токены, порт)
├── package.json
├── src/
│   ├── bot/
│   │   ├── bot.js              # Telegram бот (grammY) — команды, inline кнопки
│   │   └── ai-assistant.js     # AI-секретарь через Groq API
│   ├── db/
│   │   └── database.js         # SQLite (better-sqlite3) — все CRUD операции
│   ├── cron/
│   │   └── reminders.js        # Cron — напоминания, утренний/вечерний дайджест
│   ├── utils/
│   │   └── helpers.js          # Парсинг дат, форматирование, система ошибок
│   └── webapp/
│       ├── server.js           # Express API + Telegram WebApp авторизация
│       └── public/
│           ├── index.html      # Mini App HTML
│           ├── style.css       # Тёмная тема, адаптивный дизайн
│           └── app.js          # Фронтенд логика
├── CHEATSHEET.md               # Быстрая шпаргалка
└── ERRORS.md                   # Справочник ошибок
```

## Стек

- **Бот:** grammY (Telegram Bot API)
- **AI:** Groq API (llama-3.3-70b-versatile)
- **БД:** SQLite (better-sqlite3, WAL mode)
- **Сервер:** Express.js
- **Крон:** node-cron
- **Даты:** Luxon

## Таблицы БД

| Таблица | Описание |
|---------|----------|
| users | Пользователи (tg_id, timezone, настройки дайджестов, DND) |
| categories | Категории задач (5 дефолтных при регистрации) |
| tasks | Задачи (title, priority 1-4, status, due_date/time, subtasks) |
| recurring_rules | Правила повторений (daily/weekly/monthly) |
| reminders | Напоминания (fire_at UTC, offset_minutes, sent) |
| habits | Привычки (стрики, частота) |
| habit_log | Лог выполнения привычек |
| daily_stats | Статистика по дням |

## Команды бота

| Команда | Описание |

### Быстрое создание задач
Просто напиши текст — бот создаст задачу! Примеры:
- `купить молоко` → задача на сегодня
- `завтра встреча в 14:00` → задача на завтра с временем
- `25.03 отчёт !1` → задача на дату с высоким приоритетом

## Система напоминаний

1. **Автоматические** — за 15 мин до задачи с временем
2. **Ручные** — через кнопку "⏰ Напомнить" (5/15/30 мин, 1/3 часа, 1 день)
3. **Snooze** — отложить напоминание на 30 мин
4. **Утренний дайджест** — сводка задач утром (настраивается)
5. **Вечерний обзор** — итоги дня + перенос невыполненного

## AI-секретарь (Groq)

AI видит все ваши задачи, привычки, просроченное — и может:
- Создавать задачи через диалог
- Переносить/завершать задачи
- Давать план дня
- Разбивать большие задачи
- Давать советы по продуктивности

## Установка и запуск

```bash
# Клонировать
git clone git@github.com:alphaleaders/alpha.git
cd alpha

# Установить зависимости
npm install

# Настроить .env
cp .env.example .env
# Заполнить BOT_TOKEN, GROQ_KEY, ADMIN_ID

# Запуск
npm start

# Или в dev-режиме (auto-restart)
npm run dev
```

## Деплой на сервер (PM2)

```bash
pm2 start index.js --name alpha-planner
pm2 save
```

## Переменные окружения

| Переменная | Описание | Обязательно |
|-----------|----------|------------|
| `BOT_TOKEN` | Telegram Bot Token от @BotFather | ✅ |
| `GROQ_KEY` | Groq API ключ | Нет (AI будет выключен) |
| `WEBAPP_URL` | URL Mini App (https) | Нет |
| `PORT` | Порт сервера | Нет (по умолч. 3100) |
| `ADMIN_ID` | Telegram ID для уведомлений | Нет |
