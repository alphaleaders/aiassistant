# Alpha Planner — Справочник ошибок 🔧

## Коды ошибок приложения

| Код | Сообщение | Причина | Решение |
|-----|-----------|---------|---------|
| `TASK_NOT_FOUND` | ❌ Задача не найдена | Неверный ID или задача удалена | Проверить /today или /all |
| `USER_NOT_FOUND` | ❌ Пользователь не найден | Не зарегистрирован | Нажать /start |
| `INVALID_DATE` | ❌ Неверный формат даты | Не распознана дата | Формат: завтра, 25.03, 2026-03-25 |
| `INVALID_TIME` | ❌ Неверный формат времени | Не распознано время | Формат: 14:30, 9.00 |
| `EMPTY_TITLE` | ❌ Название пустое | Пустой текст задачи | Ввести текст |
| `CATEGORY_NOT_FOUND` | ❌ Категория не найдена | Неверный ID категории | Проверить /categories |
| `DB_ERROR` | ❌ Ошибка базы данных | SQLite ошибка | Проверить логи, перезапустить |
| `HABIT_NOT_FOUND` | ❌ Привычка не найдена | Неверный ID | Проверить /habits |
| `PERMISSION_DENIED` | ❌ Нет доступа | Чужая задача | — |
| `RATE_LIMIT` | ⏳ Слишком много запросов | Спам | Подождать |
| `WEBAPP_AUTH_FAILED` | ❌ Ошибка авторизации | Невалидный initData | Открыть из Telegram |

## Ошибки запуска

| Симптом | Причина | Решение |
|---------|---------|---------|
| `BOT_TOKEN не задан` | Пустой .env | Заполнить BOT_TOKEN в .env |
| `409: Conflict` | Бот запущен в 2х местах | Остановить другой экземпляр |
| `401: Unauthorized` | Невалидный токен | Получить новый у @BotFather |
| `SQLITE_CANTOPEN` | Нет прав на запись | `chmod 755 .` или запуск от правильного пользователя |
| `Cannot find module 'grammy'` | Не установлены зависимости | `npm install` |

## Ошибки AI (Groq)

| Симптом | Причина | Решение |
|---------|---------|---------|
| `AI assistant disabled` | Нет GROQ_KEY | Добавить в .env |
| `Timeout` | Groq не отвечает | Попробовать позже |
| `rate_limit_exceeded` | Лимит Groq API | Подождать или обновить план |
| `invalid_api_key` | Неверный ключ | Проверить GROQ_KEY |
| `model_not_found` | Модель недоступна | Проверить название модели в ai-assistant.js |

## Ошибки напоминаний

| Симптом | Причина | Решение |
|---------|---------|---------|
| Не приходят напоминания | Бот не запущен | Проверить PM2 |
| Приходят не вовремя | Неверный timezone | /timezone Europe/Moscow |
| Дубли напоминаний | 2 экземпляра бота | Остановить лишний |
| `Failed to send` | Пользователь заблокировал бота | Нормально, пропускаем |

## Ошибки WebApp

| Симптом | Причина | Решение |
|---------|---------|---------|
| Белый экран | Нет HTTPS | Настроить SSL (обязательно для Mini App) |
| 401 Unauthorized | initData невалиден | Открывать только из Telegram |
| Не загружается | WEBAPP_URL не задан | Заполнить в .env |
| CORS ошибки | Неверный домен | Проверить настройки BotFather |

## Логирование

Все ошибки выводятся в консоль с тегами:
- `[REMINDER]` — ошибки напоминаний
- `[REMINDER CRON]` — ошибки cron
- `[DIGEST CRON]` — ошибки дайджестов
- `[MORNING]` / `[EVENING]` — ошибки дайджестов
- `[AI]` — ошибки Groq API
- `[API]` — ошибки Express API
